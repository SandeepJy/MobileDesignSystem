# MobileDesignSystem

A Swift Package providing reusable UI components for iOS applications built with SwiftUI.

## Overview

MobileDesignSystem is a collection of SwiftUI components designed to provide consistent,
customizable UI elements for iOS applications. The library includes text views and input
fields with enhanced styling capabilities while maintaining native iOS behavior.

## Components

### MDSTextView

A SwiftUI text view component that supports both editable and read-only text display.

**Features:**
- Editable and static text modes
- Attributed text support
- Customizable fonts, colors, and alignment
- Scrollable and non-scrollable options
- Border and corner radius customization

### MDSInputField

A SwiftUI view that wraps a UIKit text field with enhanced styling and focus states.

**Features:**
- Placeholder text with custom color
- Customizable padding (left and right)
- Focus state with animated border color change
- Keyboard type configuration
- Secure text entry support
- Enabled/disabled states
- Text change and editing callbacks
